Detailed Description

Implements IGoR's Bayesian Network structure.

Author
Q.Marcou
Version
1.0

Implements IGoR's Bayesian Network structure through an acyclic directed graph. Together with the recombination model topology it also contains the error model. This class implements various methods to extract information from the graph structure such as the order in which RecEvents must be processed provided the topological constraints.
