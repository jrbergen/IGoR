Detailed Description

Deletion recombination event.

Author
Q.Marcou
Version
1.0

The Deletion RecEvent models deletions of one genomic fragment on a given side. Deletions can be either positive or negative (= Palindromic insertions)

By construction the corresponding GeneChoice must have been explored first.
