Detailed Description

Estimates sequences generation probability.

Author
Q.Marcou
Version
1.0

This Counter implements an estimator for the generation probability of evaluated sequences. Alternatively the counter can record the probability of generation of putative ancestor (unmutated/error free) sequences and their associated posterior probability.
