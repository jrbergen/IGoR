Detailed Description

A modified Smith-Waterman alignment class.

Author
Q.Marcou
Version
1.0

The Aligner class allows to perform SW alignments according to the parameters (substitution matrix,gap penalty) supplied upon construction of the object. The SW alignments matrix has been altered for V and J in order to allow for deletions on the deleted side only. Alignments can be made in parallel using openMP
